## 📖 Project Description

This project is an enhanced version of my Portfolio Website built using:

- HTML
- CSS
- JavaScript

The objective of this assignment was to add JavaScript interactivity to an existing HTML/CSS portfolio website.

---

## 🚀 Features Implemented

### 1️⃣ Hobby Interaction Enhancements

✔ Added an input box below the hobby description  
✔ Added a button to submit new hobby benefits  
✔ On button click:
- The input text is added as a new list item
- The input field is cleared automatically  

✔ Added a **Delete button** next to each list item  
✔ Clicking Delete removes that specific item dynamically  

---

### 2️⃣ Contact Form Interactivity

✔ Added JavaScript form handling  
✔ On form submission:
- Default submission is prevented
- An alert message is displayed confirming successful submission  

---

### 3️⃣ Dynamic Footer

✔ Footer displays:
- Current date
- Current time  


✔ Time updates automatically every second using `setInterval()`.
